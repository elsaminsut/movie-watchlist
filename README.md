# Movie Watchlist

A small Python movie app for managing a personal movie collection. 

The goal is not just to build a movie database app, but also to experiment with different programming concepts and tools in one project, including:
- Python programming
- working with SQL databases
- using external APIs
- generating HTML pages
- styling with CSS
- organizing a small multi-file project

## Project Files

- `movies.py` – main program with the menu and user interaction
- `movie_storage_sql.py` – database layer for storing and updating movie data
- `index_template.html` – HTML template used to generate the movie page
- `index_movies.html` – generated movie website
- `style.css` – styling for the movie website

## Technologies Used

- Python
- SQLite, SQLAlchemy
- HTML, CSS
- Matplotlib
- Requests

## Installation
1. Clone this repository
2. Install the required dependencies
```
pip install -r requirements.txt
```
## Usage
1. Run the script
```
python movies.py
```
2. Choose different actions from the CLI menu and execute them
